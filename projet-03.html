PROJET 03 / 14 — OpenClassrooms

Déployez un réseau local filaire et sans-fil dans le respect des bonnes pratiques

Cisco Packet TracerTCP/IP DHCPWi-Fi WPA2 EnterpriseRadius / AAA Réseau invité
// Contexte

Ce projet consistait à concevoir et déployer l'infrastructure réseau complète d'une médiathèque fictive (Valmonte Media) en environnement simulé via Cisco Packet Tracer. La solution comprend un réseau filaire pour les postes fixes et serveurs, un réseau Wi-Fi sécurisé WPA2 Enterprise pour le personnel (authentification via serveur Radius), et un réseau Wi-Fi visiteurs isolé du LAN principal.

// Compétences mises en œuvre
Cisco Packet Tracer Adressage IP / TCP/IP Serveur DHCP Serveur Radius / AAA WPA2 Enterprise (AES) WPA2 Personal Réseau invité isolé Topologie réseau
// Captures du projet
Capture 01 — Topologie réseau complète

Vue d'ensemble de l'infrastructure dans Cisco Packet Tracer : serveurs (Radius, stockage), switch central, postes filaires, routeur Wi-Fi et appareils sans-fil (laptops, tablettes, smartphones, webcam IoT). Les tests ICMP confirment la connectivité entre les segments.

Capture 02 — Configuration du serveur DHCP

Le serveur DHCP distribue automatiquement les adresses IP sur le réseau 192.168.10.0/24, avec une passerelle par défaut à 192.168.10.254 et le serveur DNS à 192.168.10.1. Le pool "serverPool" peut gérer jusqu'à 254 clients.

Capture 03 — Serveur Radius / AAA (authentification Wi-Fi personnel)

Le serveur Radius authentifie chaque appareil Wi-Fi du personnel individuellement. Chaque compte (Laptop0, Tablette0, Webcam...) dispose de ses propres identifiants, garantissant une traçabilité complète des connexions sans-fil.

Capture 04 — Sécurité Wi-Fi WPA2 Enterprise

Le réseau Wi-Fi du personnel est sécurisé en WPA2 Enterprise avec chiffrement AES, relié au serveur Radius (192.168.10.2 port 1645). Chaque connexion est authentifiée individuellement, contrairement à une clé partagée PSK.

// Ce que j'ai appris
▸Concevoir une topologie réseau complète avec Cisco Packet Tracer
▸Configurer un serveur DHCP et comprendre l'adressage IP
▸Mettre en place une authentification Wi-Fi WPA2 Enterprise via Radius
▸Isoler un réseau visiteurs du réseau interne pour la sécurité
