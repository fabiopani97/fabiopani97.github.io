PROJET 02 / 14 — OpenClassrooms

Configurez et automatisez le déploiement d'un poste de travail Windows 11

MDTWindows 11 Déploiement automatiséLite Touch Installation WinPE
// Contexte

Dans ce projet, j'ai mis en place une solution de déploiement automatisé de postes Windows 11 en utilisant Microsoft Deployment Toolkit (MDT). L'objectif était de créer une séquence de tâches complète permettant d'installer le système d'exploitation, d'appliquer les mises à jour et de déployer les logiciels métier de façon standardisée, sans intervention manuelle sur chaque machine.

// Compétences mises en œuvre
Microsoft Deployment Toolkit (MDT) Windows PE (WinPE) Lite Touch Installation Séquence de tâches Windows Update automatisé Déploiement applicatif VirtualBox
// Captures du projet
Capture 01 — Installation du système d'exploitation

La séquence MDT "Lite Touch Installation" installe Windows 11 automatiquement sur le poste cible. La barre de progression montre l'avancement de l'action "Install Operating System" à 71%.

Capture 02 — Application automatique des mises à jour

Après l'installation de l'OS, MDT télécharge et applique automatiquement les correctifs de sécurité Windows (ici le KB5079473 de mars 2026), sans aucune intervention manuelle.

Capture 03 — Déploiement réussi

Le rapport final "Deployment Summary" confirme que le déploiement s'est terminé avec succès : 0 erreur, 0 avertissement. Le poste est prêt à l'emploi avec tous les logiciels métier (VLC, Google Chrome, AnyDesk...).

// Ce que j'ai appris
▸Créer et configurer un serveur MDT avec un partage de déploiement
▸Construire une séquence de tâches automatisée (OS + MAJ + logiciels)
▸Générer un média de démarrage WinPE pour le boot réseau
▸Réduire drastiquement le temps de déploiement par rapport à une installation manuelle
