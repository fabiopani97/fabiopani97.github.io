PROJET 04 / 14 — OpenClassrooms

Reconditionnez un parc d'ordinateurs et industrialisez sa configuration

FOG ProjectPartclone Clonage disqueActive Directory Jonction domaineVirtualBox Image maître
// Contexte

Ce projet consistait à industrialiser le déploiement d'un parc de postes Windows 11 pour un cybercafé fictif via FOG Project, un serveur de clonage open-source. Une image maître Windows 11 a été capturée, stockée sur le serveur FOG, puis déployée sur plusieurs machines virtuelles. Les postes déployés sont ensuite joints au domaine Active Directory "CYBERCAFE" pour une gestion centralisée.

// Compétences mises en œuvre
FOG Project (serveur de clonage) Partclone Capture d'image maître Déploiement multicast Active Directory Jonction au domaine VirtualBox Linux (administration FOG)
// Captures du projet
Capture 01 — Serveur FOG : image maître Windows 11

L'interface web de FOG Project montre l'image maître "Windows11-Master-1" (27.72 GiB, format ZSTD compressé) capturée le 25 avril 2026. Cette image servira de base pour déployer tous les postes du parc de façon identique.

Capture 02 — Restauration d'image via Partclone

Partclone restaure l'image Windows 11 sur un nouveau poste à 1.53 GB/min. Le processus copie 27.5 GB de données sur le disque cible (/dev/sda3) depuis le serveur FOG (192.168.10.10). À 89% d'avancement, il ne reste qu'environ 2 minutes.

Capture 03 — Jonction au domaine Active Directory

Après le déploiement, le poste est joint au domaine "CYBERCAFE". L'écran de connexion affiche "Connectez-vous à CYBERCAFE", confirmant que la machine est bien membre du domaine et peut être gérée centralement via Active Directory.

Capture 04 — Postes visibles dans Active Directory

La console "Utilisateurs et ordinateurs Active Directory" sur le domaine cybercafe.lan montre les 4 postes déployés et joints au domaine : PC-W11-01, PC-ADMIN-CYBER, PC-W11-02 et PC-W11-03. Le parc est désormais géré centralement.
